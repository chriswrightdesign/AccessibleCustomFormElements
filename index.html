Custom form elements

A skeleton collection of form elements created to work across browsers and devices, and maintain accessibility by still allowing keyboard control and focus, and work relatively well with screen readers.

This project is in its early alpha stages, at the moment I'm building a series of different form controls that I'll then go through testing with. Use these controls at your peril. Test test test.

This implementation uses two anchor links to control the stepper, alternatively you could use radio buttons and add some keyboard support, however, it wouldn't work so well repeatedly hitting the same radio button.

You need to detect a form of touch, as you'll end up with double tap zooms on mobile devices. Keep touch target sizes in mind, as it's quite easy to hit the number input if you have giant sausage fingers and the buttons are too small.
